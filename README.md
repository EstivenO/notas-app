# 📚 Gestión de Materias - JavaScript

Aplicación web desarrollada con JavaScript puro para gestionar materias académicas.

## 🚀 Funcionalidades

- ✅ Crear materias
- ✅ Editar materias
- ✅ Eliminar materias
- ⭐ Marcar como importantes
- 💾 Persistencia con localStorage

## 🧠 Tecnologías

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 🎯 Objetivo

Este proyecto fue desarrollado como práctica para consolidar conocimientos en:

- Manipulación del DOM
- Manejo de estado
- Eventos
- Persistencia en el navegador
